wgs84Ellipsoid

PURPOSE

wgs84Ellipsoid generate a WGS84 ellipsoid parameter struct

SYNOPSIS

function E = wgs84Ellipsoid(lengthUnit)

DESCRIPTION 

wgs84Ellipsoid  generate a WGS84 ellipsoid parameter struct

 Inputs
 ------
 lengthUnit: currently not implemented, for compatibility either nargin=0 or 'm' or 'meter' is accepted without conversion

 Outputs
 ------
 E: struct with referenceEllipsoid() properties

CROSS-REFERENCE INFORMATION

This function calls:
  • getreferenceEllipsoid getreferenceEllipsoid Select available ellipsoid
This function is called by:
  • aer2ecef aer2ecef convert azimuth, elevation, range to target from observer to ECEF coordinates
  • ecef2geodetic ecef2geodetic convert ECEF to geodetic coordinates
  • geodetic2ecef geodetic2ecef convert from geodetic to ECEF coordiantes
